# 需要什么？
* python 3
* numpy (pip install numpy)
* selenium (pip install selenium)
# 如何使用?
* step1
<code>git clone https://github.com/WinstonDeng/sudo_online.git</code>
* step2
<code>cd sudo_online</code>
* step3
<code>python bingo_sudo_online.py</code>
